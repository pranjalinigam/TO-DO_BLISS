# 🌸 Todo Bliss

> **A modern all-in-one productivity dashboard to help you organize tasks, build habits, stay focused, and achieve your goals.**

---

## ✨ Overview

Todo Bliss is a beautifully designed productivity web application that combines task management, calendar planning, habit tracking, focus sessions, notes, goals, analytics, and more into a single intuitive dashboard.

Designed with a soft pastel aesthetic and responsive layout, it provides users with a clean and enjoyable productivity experience.

---

## 🚀 Features

### 🏠 Home
- Personalized welcome section
- Daily productivity overview
- Quick Add Task
- Progress summary
- Upcoming tasks
- Motivational quotes

### 📋 Task Management
- Add, edit, and delete tasks
- Categorize tasks
- Mark tasks as completed
- Task statistics
- Search & filtering

### 📅 Calendar
- Monthly calendar view
- Navigate between months
- Highlight current day
- View tasks by date

### ⏳ Focus Timer
- Pomodoro Timer
- Start, Pause & Reset
- Session tracking

### 💧 Water Tracker
- Daily hydration tracking
- Progress indicator
- Goal completion

### 🌱 Habit Tracker
- Track daily habits
- Habit streaks
- Progress monitoring

### 📝 Notes
- Create personal notes
- Edit & delete notes
- Organized workspace

### 🎯 Goals
- Set personal goals
- Track progress
- Goal completion monitoring

### 📊 Analytics
- Productivity insights
- Completion statistics
- Daily progress overview

### ⚙️ Settings
- User preferences
- Theme settings
- Application customization

### 🌙 Dark Mode
- Fully functional Dark/Light mode
- Smooth theme switching
- Theme preference saved using Local Storage

# 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/pranjalinigam/TO-DO_BLISS.git
```

### Navigate to the Project

```bash
cd TO-DO_BLISS
```

### Install Dependencies

```bash
npm install
```

### Run the Project

Open `index.html` in your browser

or

```bash
npm start
```

(if configured)
